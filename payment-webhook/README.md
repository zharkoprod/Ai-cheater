# Постбэки об оплате → уведомления в Telegram (Нейро-ЧИТЕР)

Маленький облачный «приёмник», который ловит уведомления об оплате от **ЮKassa**
и присылает тебе сообщение в **Telegram**:

```
Лендинг → кнопка → платёжное окно ЮKassa → оплата
                                              │
                                ЮKassa шлёт постбэк (HTTP-уведомление)
                                              │
                                  этот Worker (Cloudflare, бесплатно)
                                              │
                                  бот пишет тебе в чат:
                                  «✅ Новая оплата 2 990 ₽ — тариф Базовый»
```

Лендинг трогать не надо — кнопки уже ведут на платёжные ссылки ЮKassa.

---

## Что понадобится (3 вещи)

1. **Токен Telegram-бота** — от [@BotFather](https://t.me/BotFather): `/newbot` → пришлёт токен вида `12345:AAH...`
2. **chat_id** — куда слать уведомления (см. шаг 2 ниже)
3. **shopId и секретный ключ ЮKassa** — Личный кабинет ЮKassa → **Интеграция → Ключи API**

> ⚠️ Эти данные — пароли. Они задаются как «секреты» в Cloudflare и **не попадают в репозиторий**.

---

## Шаг 1. Создать бота

1. Открой [@BotFather](https://t.me/BotFather) → `/newbot` → задай имя и username.
2. Сохрани **токен**.

## Шаг 2. Узнать chat_id (куда слать уведомления)

**Вариант «в личку себе»:**
1. Напиши что-нибудь своему новому боту (нажми Start).
2. Открой в браузере (подставь свой токен):
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
3. Найди `"chat":{"id":123456789` — это твой `chat_id`.

**Вариант «в группу/канал»:**
1. Добавь бота в группу/канал и сделай его администратором.
2. Напиши в группу любое сообщение и снова открой `/getUpdates` — там будет
   `chat_id` группы (обычно с минусом, например `-1001234567890`).

## Шаг 3. Задеплоить Worker на Cloudflare

### Способ A — через терминал (рекомендуется)
```bash
npm install -g wrangler
wrangler login
cd payment-webhook

# задаём секреты (значения вставишь по запросу, в файлы они не пишутся)
wrangler secret put TELEGRAM_BOT_TOKEN
wrangler secret put TELEGRAM_CHAT_ID
wrangler secret put YOOKASSA_SHOP_ID
wrangler secret put YOOKASSA_SECRET_KEY

wrangler deploy
```
После деплоя Wrangler покажет адрес вида
`https://neuro-cheater-payment-webhook.<твой-аккаунт>.workers.dev` — это и есть
URL для постбэков.

### Способ B — через панель Cloudflare (без терминала)
1. [dash.cloudflare.com](https://dash.cloudflare.com) → **Workers & Pages** → **Create** → **Create Worker**.
2. Назови, нажми **Deploy**, затем **Edit code** → вставь содержимое `worker.js` → **Deploy**.
3. **Settings → Variables and Secrets** → добавь 4 секрета:
   `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `YOOKASSA_SHOP_ID`, `YOOKASSA_SECRET_KEY`.
4. Скопируй адрес Worker'а (`*.workers.dev`).

## Шаг 4. Указать адрес постбэка в ЮKassa

1. Личный кабинет ЮKassa → **Интеграция → HTTP-уведомления**.
2. В поле URL вставь адрес Worker'а из шага 3.
3. Включи событие **`payment.succeeded`** (успешная оплата). Сохрани.

## Шаг 5. Проверка

1. Открой адрес Worker'а в браузере — должно ответить
   `Neuro-Cheater payment webhook is alive`.
2. Сделай тестовую оплату на лендинге (можно на минимальную сумму /
   тестовый магазин ЮKassa) — в Telegram должно прийти уведомление.

## Частые вопросы

**Пришло «not verified»?** Значит платёж не подтвердился через API ЮKassa —
проверь, что `YOOKASSA_SHOP_ID` и `YOOKASSA_SECRET_KEY` корректны и взяты из
того же магазина, где настроены платёжные ссылки.

**Не приходит сообщение в Telegram?** Проверь `chat_id` и что ты нажал Start
у бота (бот не может писать первым тому, кто его не запускал).

**Дубли уведомлений?** ЮKassa может повторять постбэк, если не получила
ответ `200`. Worker всегда отвечает `200`, так что дублей быть не должно;
если появятся — добавим защиту от повторов через Cloudflare KV.

**Какие суммы → какой тариф.** Сопоставление заметно в начале `worker.js`
(блок `TARIFFS`). Если поменяешь цены — поправь там же.
